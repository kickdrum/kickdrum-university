Cygwin Setup steps

  1. Download and run the installer https://www.cygwin.com/install.html ↗

  2. In the Installation Wizard progess forward with default settings to the Mirror Selection Step (choose the one shown in the image) -

  3. Post this select All packages for the base installation.

  4. Once all of this is done Cygwin should be installed and ready to go. You should see an icon for launching Cygwin Terminal on your desktop.

  5. There's one more step before you start using it. To be able to access your Desktop folder from the Cygwin Terminal run the following commands -

# Sets up Desktop folder path
 cygpath -u 'C:\Users\<YOUR_WINDOWS_USERNAME>\Desktop'
#  cd into it
 cd C:/Users/<YOUR_WINDOWS_USERNAME>/Desktop
# Create a directory and check if it reflects on your desktop
 mkdir hello

At the end it should look something like this -

And voila! You can have the same awesome UNIX CLI experience on your Windows machine now.

You can search whatever package ↗ you like and install it.

Installing new cygwin packages requires you to run the installer again with default settings up until the package selection step.
